Rhetorical Analysis Part 2: Parallelism

Posted by Brendan Caporaletti
May 13, 2016

“I think that the American people are never going to elect a president who insults Mexicans, who insults Muslims, who insults women, who insults African-Americans.” -Bernie Sanders, 2016

We see parallel structure used all over candidates debates and speeches. It’s a powerful tool to reiterate and build a point using the repetition of singular pattern. We wanted to figure out a way to identify these classic uses of parallel structure like you see in the quote above. So to define parallel structure, it’s simply the adjacent repetition of an identical grammatical structure. So the first thing we must do is identify the grammatical structure In order to accomplish this we used nltk’s part of speech tagger, which uses the Penn Treebank Part of Speech Tags, listed here.

Once tagged we can start to identify these grammatical patterns. In order to save time later we pre-process a list of the locations of every existent grammatical pattern. The length of phrase is adjustable, but patterns under 3 words yield too many false-positive and patterns over 7 words rarely actually exist. So we build a dictionary for every document where keys are every single present 3 to 7 word grammatical pattern and the value being a list of every index within the document where that pattern appears. We serialized and stored this dictionary such that it never had to be recomputed.

With this dictionary of pattern references we can start looking for repetition in these patterns. We took a straightforward approach by simply looking for two, three, or four instances of the same pattern that are directly adjacent to one another. This yields lot’s of great examples:

“we want peace we want freedom we want security” - Mitt Romney, 2012

“the power companies the oil companies the drug companies” - John Kerry, 2004

“they solved their problems they confronted their challenges they embraced their opportunities “ - Jimmy Carter, 1980
